regionModel command :h3

NOTE: In the current CFDEMcoupling version, this model is no longer used.

Defined in "couplingProperties"_CFDEMcoupling_dicts.html#couplingProperties
dictionary.

[Syntax:]

regionModel model; :pre

model = name of the regionModel to be applied :ul

[Examples:]

regionModel allRegion; :pre

NOTE: This examples list might not be complete - please look for other models
(regionModel XY) in this documentation.

[Description:]

The {regionModel} is the base class for region models to select a certain region
for coupled simulation.

[Restrictions:]

none

[Default:]

none
